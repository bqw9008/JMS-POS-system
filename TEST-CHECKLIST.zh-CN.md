# POS 系统测试清单

这份清单用于当前 WPF 桌面版的手工验收。

完成后可直接把对应项目勾成 `[x]`。

## 范围

当前重点覆盖：

- WPF 主壳启动
- 中英文切换
- 分类、商品、库存、收银、销售记录、报表、设置流程
- 结算后的小票预览窗口

当前暂不覆盖：

- 真实小票打印机接入
- 扫码枪硬件兼容性
- 登录和角色
- 退款

## 测试准备

环境：

- Windows
- .NET 9 SDK

建议准备步骤：

1. 执行 `dotnet build`
2. 如需更快验证主流程，可先插入测试数据：

```powershell
$env:POS_SEED_TEST_DATA = "1"
dotnet run
```

3. 做正常使用验证时，取消这个环境变量或将其设为 `0`
4. 如果要重测数据库首次初始化，可先删除本地 `pos.db`
